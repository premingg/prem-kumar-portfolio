# Prem Kumar — Portfolio

Personal developer portfolio built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- Supabase
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- 3D animated avatar entry screen
- Parallax star-field background
- Scroll-snap project showcase
- GitHub contribution graph
- Custom smooth cursor
- Fully responsive
